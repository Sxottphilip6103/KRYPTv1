System Mechanics Manual

Krypt is a local-first, zero-database key derivation tool. Instead of storing your passwords, it dynamically computes them using mathematics.

By blending three independent secrets with a specific site identifier, it achieves zero-knowledge security:

  • No cloud tracking or remote data storage.
  • Identical inputs consistently produce identical outputs.
  • Splitting variables across 3 inputs removes any single master security vulnerability.
